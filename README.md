# For Noor

A static animated apology website from Ranvir to Noor.

## Run locally

Open `index.html` directly in a browser. No server, build tools, package manager, backend, or database are required.

## Edit content

Open `index.html` and search for `EDIT`. Each customizable message, card, promise, memory, and final note is marked with an HTML comment.

## Deploy to GitHub Pages

1. Create or open the GitHub repository.
2. Push these files to the `main` branch.
3. In GitHub, go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save. The site will be available at:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/
```

This project is safe for GitHub Pages because it is only HTML, CSS, and vanilla JavaScript with CDN imports.
